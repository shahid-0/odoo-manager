# Security Policy

## Supported Versions

Only the latest version of Odoo Manager is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Odoo Manager, please send an e-mail to your-email@example.com. All security vulnerabilities will be promptly addressed.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
